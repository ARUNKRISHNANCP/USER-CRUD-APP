# USER-CRUD-APP
A React-based CRUD application that allows users to Create, Read, Update, and Delete user records using a Fake REST API powered by JSON Server. The project demonstrates React Hooks, React Router, Axios API integration, and responsive UI design with Bootstrap. Perfect for learning frontend development and REST API interactions.


User List CRUD Application Using React and Fake JSON REST API

The User List CRUD Application is a web-based application developed using React.js that performs basic CRUD (Create, Read, Update, Delete) operations on user data. The application uses JSON Server as a fake REST API to simulate backend functionality, allowing users to manage records without requiring a real database.

The system provides an intuitive interface where users can add new records, view user details, edit existing information, and delete records from the list. All data is stored in a local db.json file and accessed through RESTful API endpoints generated by JSON Server.

The application demonstrates core React concepts such as components, state management, hooks (useState and useEffect), routing with React Router, form handling, API integration using Axios, and dynamic rendering of data in a table format.

This project is ideal for learning frontend development and understanding how React applications interact with REST APIs. It serves as a practical example of building a complete single-page application (SPA) with real-world CRUD functionality.

Key Features
Create new user records
Display users in a tabular format
View detailed user information
Update existing user records
Delete user records
REST API integration using Axios
Navigation using React Router
Responsive user interface with Bootstrap
Fake backend using JSON Server

Objective

The main objective of this project is to understand and implement CRUD operations in React by consuming RESTful APIs and managing application state efficiently while providing a user-friendly interface for data management.
